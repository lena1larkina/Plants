Это приложение на сервлетах и JSP, позволяющее пользователям управлять коллекцией растений с оценкой их редкости.
Приложение обеспечивает авторизацию пользователей и работу с таблицей растений.
В базе данных 2 таблицы: profile - хранит данные о зарегистрированных пользователях: id_profile, login, password_hash) rare_plants - таблица с растениями,
с которыми может работать пользователь. Реализованы CRUD операции: создание, чтение, обновление, удаление. Внешний ключ id_profile связывает растения с текущим пользователем.
Каждый пользователь видит и редактирует только свои записи в таблице rare_plants.
После запуска приложения открывается страница входа. После авторизации пользователь попадает на главную страницу "Мои растения",
где видит таблицу со своими растениями, содержащую колонки: ID, Название и Оценка редкости.
Сверху расположены кнопки для добавления нового растения и поиска растений по имени.
Для каждой записи в таблице доступны действия "Редактировать" и "Удалить", позволяющие модифицировать или удалять растение из коллекции.
В правом верхнем углу страницы находится кнопка "Выйти" для завершения сессии.
